# ge-package — Title & Thumbnail Packager (Step 3 of 8)

## STEP 0 — INTAKE HARD STOP
Confirm: the **topic/title direction**, **primary keyword**, **format**, and the
**core emotion/payoff** of the video. If a script exists, read it to pull the
single strongest "moment" for the thumbnail.

## STEP 0.5 — LOAD SHARED PROTOCOL
Read `ge-compliance/SKILL.md`. Titles and thumbnail text must contain NO banned
words ("best/expert/guarantee/win," etc.) and imply no outcome.

## STEP 1 — TITLES (3 variants)
Each title: keyword **front-loaded**, **<60 characters**, blends keyword +
emotion/clarity, mobile-legible. Label each variant by angle:
- **V1 — Search/keyword-led** ("[CITY] Car Accident Settlement: What to Expect")
- **V2 — Curiosity/emotion-led** ("The Mistake That Tanks Most Injury Claims")
- **V3 — Specific/number-led** ("3 Things to Do Before You Talk to an Insurer")
Note character count for each.

## STEP 2 — THUMBNAIL BRIEFS (3 concepts)
For each concept output:
```
Concept <n>
Visual:        <subject + setting; a real face showing one clear emotion>
Emotion:       <concern / surprise / determination — one>
Text overlay:  <<=3 words, high impact, NO banned words>
Color palette: <2–3 hex codes, high contrast>
Composition:   <left/right face, where text sits, focal point>
Mobile check:  <is text legible at thumbnail size? yes/why>
Pairs with:    <which title variant it best supports>
```
Design rule: thumbnail must be accurate to the content (YouTube Test & Compare
optimizes for watch-time share, so clickbait that misleads loses).

## STEP 3 — SYNERGY CHECK
Confirm the chosen title + thumbnail don't repeat the same words (they should add
information to each other, not echo). Recommend the strongest title↔thumbnail pair.

## STEP 4 — A/B TEST PLAN
Recommend loading 2–3 thumbnail variants into YouTube's native **Test & Compare**
for any video expected to clear ~1,000 impressions; specify which variants and
what hypothesis each tests (e.g., "face vs. document," "red vs. blue ground").

## STEP 5 — OUTPUT + HANDOFF
Deliver titles + thumbnail briefs + synergy pick + A/B plan + the COMPLIANCE CHECK.
> **Next step:** Run **ge-seo** for the description/metadata, then **ge-publish**
> for the upload checklist.
