# ge-shorts — Shorts Repurposer (Step 5 of 8)

## STEP 0 — INTAKE HARD STOP
Confirm: the **long-form transcript or script** (or the clip-able-moments list
from ge-script), the **full video title/URL** to funnel toward, and the **city**.
If only a topic is given with no source, ask for the transcript first.

## STEP 0.5 — LOAD SHARED PROTOCOL
Read `ge-compliance/SKILL.md`. Each Short needs a brief on-screen "not legal
advice" caption (condensed Disclaimer A) and zero banned language. Reaction Shorts
also carry the condensed commentary disclaimer (C).

## STEP 1 — SELECT MOMENTS
Pick the **3–5 strongest self-contained moments** (surprising fact, common
mistake, one clear tip, a myth-bust). Each must stand alone in <45 seconds.

## STEP 2 — BUILD EACH SHORT
Output per Short:
```
SHORT <n> — <working title>
Source timestamp: <mm:ss–mm:ss from the long-form>
Hook (0–2s):      <pattern-breaking first line; text + spoken>
Body (≈10–35s):   <one insight; plain language>
Tease/funnel:     <leave it intentionally incomplete → "full breakdown in the
                   video linked in the comments/description">
On-screen captions: <key lines to burn in>
Disclaimer caption: <condensed "general info, not legal advice">
Caption/desc text:  <1–2 lines + 3 hashtags incl. one geo>
Funnel link target: <full video title/URL>
Length:           <15–45s>
```

## STEP 3 — FUNNEL DESIGN
Remember: ~74% of Shorts views come from non-subscribers, and Shorts/long-form run
on decoupled algorithms. So each Short must (a) deliver value fast and (b) create
an open loop that only the long-form closes. Recommend pinning a comment with the
long-form link and adding it to the Short's description.

## STEP 4 — OUTPUT + HANDOFF
Deliver the 3–5 Shorts + a suggested **posting order/spacing** (e.g., 1 Short/day
across the week the long-form drops) + the COMPLIANCE CHECK.
> **Next step:** Run **ge-publish** for each Short and the long-form.
